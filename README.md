# Game Engine — Collision Module ⚙️

A custom game engine module focused on collision detection and resolution, built in C++. Developed as part of my game engine programming coursework at the University of Brighton using the XCube2D framework.

## About

This project involved extending an existing 2D game engine framework to implement robust collision detection. The work required a solid understanding of engine architecture, physics fundamentals, and how to write performant, reusable systems that other game code can build upon.

## Features

- Axis-Aligned Bounding Box (AABB) collision detection
- Collision response and resolution
- Integration with the XCube2D game engine framework
- Pong-style demo game to validate the collision system

## Tech

- **Language:** C++
- **Framework:** XCube2D
- **IDE:** Visual Studio

## How to Run

1. Clone the repository
2. Open the solution file in Visual Studio
3. Build and run to see the collision demo
